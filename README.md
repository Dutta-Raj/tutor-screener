# Cuemath AI Tutor Screener

An AI-powered interviewer that conducts voice conversations with tutor candidates.

## Live Demo
[Live URL will be added after deployment]

## Features
- Voice input & output (Web Speech API)
- 5-dimension assessment (1-10 scale)
- Interview history with delete
- Downloadable report cards
- 15-minute timer with progress

## Tech Stack
- Next.js 14, TypeScript
- Groq API (Llama 3.3)
- Web Speech API

## Setup
1. Clone: `git clone https://github.com/Dutta-Raj/tutor-screener.git`
2. `npm install`
3. Create `.env.local` with `GROQ_API_KEY=your_key`
4. `npm run dev`

## Deployment
Deployed on Vercel
